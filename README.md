# Secure IoV Authentication System

A blockchain-based Secure Internet of Vehicles (IoV) authentication system with trust management and attack detection.

## Features
- Vehicle Registration
- Secure Vehicle Authentication
- Blockchain Authentication Logs
- Attack Simulation
- Trust Score Management
- Vehicle Blacklisting
- Security Dashboard
- Blockchain Visualization

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS

## Attack Types
- Sybil Attack
- Replay Attack
- DoS Attack

## Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Open:

http://127.0.0.1:5000
